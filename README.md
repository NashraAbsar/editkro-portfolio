# Editkaro.in Portfolio Showcase

A professional, responsive portfolio website built to showcase creative video edits and social media management projects. This project serves as a dynamic landing page to convert viewers into potential clients by demonstrating versatility across various video genres.

## 🚀 Key Features

* **Dynamic Video Filtering:** Easily navigate between different video categories including Short-form, Long-form, Gaming, Ads, Anime, Football Edits, Documentaries, and Color Grading.
* **Interactive UI:** Modern dark themed design featuring interactive thumbnails that link directly to YouTube for high-quality playback.
* **Fully Responsive:** Optimized for a seamless experience across desktop and mobile devices.
* **Fast Loading:** Built with a focus on performance, ensuring quick access to high-definition video content.

## 🛠️ Built With

* **HTML5** - For structured content.
* **CSS3** - For styling, layouts, and interactive filtering effects.
* **JavaScript** - To power dynamic category filtering and smooth UI transitions.

## 📂 Project Structure

The portfolio is organized by specialized editing niches to highlight specific skill sets:

* **Short-form Content:** High-energy edits for TikTok and Instagram Reels.
* **Long-form Content:** Narrative-driven editing for platforms like YouTube.
* **Specialized Edits:** Dedicated sections for niche markets like Gaming Montages, Football skills, and Documentary masterclasses.
